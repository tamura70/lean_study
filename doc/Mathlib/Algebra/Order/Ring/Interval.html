    Intervals of integers in strict ordered rings #

    These statements could perhaps be generalized, or there could be other variations provided (e.g., for ℕ instead of ℤ, or a version for locally finite SuccOrders with strictly monotone functions), but for now these are the ones that have found utility in practice (e.g., for lemmas about Real.Angle).

    k ∈ Finset.Icc m n

    Let k : ℤ. If its multiple of r > 0 in a strict ordered ring lies strictly between multiples r * (m - 1) and r * (n + 1), then m ≤ k ≤ n.
